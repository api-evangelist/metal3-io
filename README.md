# Metal3 (metal3-io)
Metal3 is a CNCF incubating project for bare metal host provisioning in Kubernetes. It uses Ironic for hardware management and integrates with Cluster API for Kubernetes-native lifecycle management of physical servers.

## APIs

### Metal3 BareMetalHost API
Metal3 extends Kubernetes with BareMetalHost CRDs for hardware inventory, firmware configuration, BIOS settings, RAID configuration, and OS provisioning of physical servers.
